# Dark---White-Theme
Projeto de frontend simples que implementa uma lista de tarefas estática com suporte a troca de tema entre “light” e “dark”.

# Projeto Bradesco - Task List

Este é um projeto frontend simples que demonstra a criação de uma lista de tarefas estática com suporte à troca de tema (light/dark).

## Funcionalidades

- Página em `HTML` com estrutura básica de título, lista de tarefas e botão de alternância de tema
- Estilos em `CSS` usando variáveis personalizadas e classes de tema
- Script em `JavaScript` para alternar entre os temas `light` e `dark`
- Atualização instantânea do visual sem recarregar a página
- Compatível com navegadores que suportam JavaScript

## Arquivos

- `index.html` - estrutura da página
- `main.css` - estilos e temas
- `app.js` - lógica de alternância entre temas

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Clique no botão de tema para alternar entre os modos claro e escuro

## Observações

- O projeto é ideal para aprendizado de manipulação do DOM, classes CSS e temas dinâmicos
- Não há dependências externas


## Créditos

Desenvolvido como um pequeno projeto de frontend para prática de HTML, CSS e JavaScript.
